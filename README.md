# BigKindsParser

> BigKindsParser는 한국 언론의 빅데이터 저장소인 BigKinds에서 추출한 데이터를 low-Code로 분석할 수 있게 만든 툴입니다.

## Requirements

- Python >= 3.8

## Installation

- macOS

  ```bash
  python3 -m pip install BigKindsParser
  ```

- Windows

  ```bash
  python -m pip install BigKindsParser
  ```
